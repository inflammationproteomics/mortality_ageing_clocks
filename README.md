# Sex-specific inflammatory and proteomic ageing clocks reveal molecular heterogeneity in biological ageing and improve mortality prediction

Code files in this repository cover the main analyses:

1. Data extraction, quality control, and baseline characteristics;

2. Identification of sex-specific inflammatory and proteomic biomarkers associated with 10-year all-cause mortality risk;

3. Biological pathway analyses of mortality-associated protein signatures;

4. Prediction of 10-year all-cause mortality using inflammatory and proteomic biomarkers;

5. Development and validation of inflammation- and proteomics-based mortality risk ageing clocks;

6. Associations of biological age acceleration with all-cause and cause-specific mortality;

7. Interactive Shiny web application for individualized biological age estimation.
